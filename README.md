# AI Medical Chatbot for Heart Disease Detection

## Overview

This is a beginner-level AI Medical Chatbot built using Python, Flask, and Hugging Face Transformers. The application allows users to upload heart-related medical images and receive an AI-generated prediction along with simple medical guidance and recommendations.

## Features

* Upload heart-related images (JPG, JPEG, PNG)
* AI-powered image classification using Hugging Face
* Medical chatbot-style explanations
* Confidence score display
* Simple and user-friendly interface

## Technologies Used

* Python
* Flask
* Hugging Face Transformers
* HTML, CSS
* PIL (Pillow)

## Installation and Setup

### Step 1: Clone Repository

git clone YOUR_GITHUB_REPOSITORY_URL

cd CHATBOT

### Step 2: Create Virtual Environment

Windows:

python -m venv venv

### Step 3: Activate Virtual Environment

Windows PowerShell:

venv\Scripts\Activate.ps1

### Step 4: Install Dependencies

pip install -r requirements.txt

### Step 5: Run Application

python app.py

### Step 6: Open Browser

http://127.0.0.1:5000

## Usage

1. Open the application.
2. Upload a heart-related image.
3. Click "Analyze Image".
4. View prediction results, confidence score, and chatbot recommendations.
